| Dossier         | Rôle                                                                  |
| --------------- | --------------------------------------------------------------------- |
| **config**      | Contient la configuration de MongoDB.                                 |
| **models**      | Contient les modèles Mongoose (collections MongoDB).                  |
| **routes**      | Définit les routes de l'API Express.                                  |
| **controllers** | Contient la logique métier (enregistrement, lecture, suppression...). |
| **serial**      | Gère la communication avec l'Arduino via le port série.               |
| **server.js**   | Point d'entrée de l'application.                                      |
| **.env**        | Stocke les informations sensibles (URI MongoDB, port...).             |
